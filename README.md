# lab1PG15012

Proyecto de ejemplo en Python para el laboratorio 1 de PG15012.

## Descripción

Este repositorio contiene el código y los recursos necesarios para el primer laboratorio del curso. Incluye:
- Scripts en Python
- Configuración básica del proyecto
- Archivos de prueba y documentación mínima

## Instalación

1. Crear un entorno virtual:
   ```bash
   python3 -m venv .venv
   ```
2. Activar el entorno:
   ```bash
   source .venv/bin/activate
   ```
3. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

Ejecutar el script principal o los módulos según el formato del proyecto:
```bash
python make_datase.py
python montecarlo.py
```
